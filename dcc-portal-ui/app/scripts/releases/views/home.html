Projects

A Cancer Project is characterized by a catalogue of genomic abnormalities (somatic mutations, abnormal expression of genes, epigenetic modifications) affecting Donors. The Cancer Projects page provides the list of all the Cancer Projects involved in the International Cancer Genome Consortium (ICGC). Users can use filters to narrow down to projects of interest then navigate through the portal to get more information about the Donors, Genes, Mutations and Files. The Cancer Projects page also provide some visualizations such as the the top mutated cancer genes across the ICGC projects, the donor distribution (number of cases for each primary site and project) and the number of somatic mutations in donor's exomes across cancer projects.

Search

The Advanced Search page allow users to explore the ICGC harmonized data through 3 interdependent and ranked lists: Donors, Genes and Mutations. Users can use filters on Donors, Genes and Mutations to narrow down his search. The Advanced Search page provides links to various analyses / visualizations that can be run based on the data of interest: Oncogrid, Enrichment Analysis, Save/edit sets, Genome Browser, Download Donor Data, View Data Files.

Analysis

The Data Analysis page allows users to launch various analyses using donor, gene, mutation or file sets created based on Data Portal search. The Data Analysis page provides analysis tools such as:

  • Enrichment Analysis: identify statistically significantly over-represented groups of gene sets when comparing user input genes and genes from a chosen background.
  • Set Operation: Set Operations on donor, gene, mutation or file sets from ICGC data.
  • Oncogrid: Oncogrid view of mutation frequency.
  • Cohort Comparison: Survival plots for 2 custom cohorts and phenotype comparison.

Releases

A DCC data release is a curated data set produced from clinical and experimental data submitted through the ICGC Data Submission site. Such data releases happen on average three times a year and are cumulative in nature. Each release is denoted by a number, such as ICGC 20 or Release 20. The DCC Data Releases page is where you can browse and download archive data files for every DCC data release in the past.

Files

The ICGC Portal is a search tool aggregating metadata about clinical, raw and experimental data. The data is actually housed on separate repositories belonging to ICGC member projects (e.g. Collaboratory - Toronto, GDC - Chicago, PCAWG - London...). The Data Repositories page allows users to browse the federated metadata, providing diverse filters to help refine searches. Once the user gets the list of files of interest, he can download a manifest that can be used with the icgc-get download tool to download the actual files stored on the various repositories or he can get the manifests used by the repository-specific download clients. This page also provides visualization such BAM or VCF statistics.
