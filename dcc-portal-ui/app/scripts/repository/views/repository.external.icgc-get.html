icgc-get

icgc-get is a universal download client for accessing ICGC data residing in various data repositories. See the User Guide for more advanced options.
For information on how to obtain access, see the Repositories documentation.

Your file selection icgc-get manifest ID is:

To download your current file selection, copy the icgc-get manifest ID below and execute the following command:
./icgc-get download -m {{vm.manifestId}}
