icgc-get

icgc-get is a universal download client for accessing ICGC data residing in various data repositories.
To download your current file selection, simply copy the icgc-get manifest ID below and execute the following command:
./icgc-get download -m {{vm.manifestId}}
See the User Guide for more advanced options.
