The Cancer Genome Collaboratory (or simply the Collaboratory) is an academic research cloud being built by the Ontario Institute for Cancer Research (OICR). The Collaboratory hosts ICGC BAM files, compute capabilities, and Docker-based analysis tools for use by its research community. Although its access is currently restricted to a small number of beta-testers, OICR plans to scale up the hardware infrastructure,increase the number of ICGC datasets available, and open the cloud to a broader community of researchers by the end of 2016. For more information, please contact help@cancercollaboratory.org .

The Collaboratory currently contains a growing repository of ICGC alignment and variant data sourced from the ICGG PanCancer Analysis of Whole Genomes project (PCAWG). The content is maintained in collaboration with the ICGC Data Coordination Center (DCC). Our user guide provides step-by-step instructions on how to access and analyse ICGC data on the Collaboratory.

The data produced by the PCAWG project, including somatic mutations, is subject to a publication ebargo. Please see the PCAWG publication and embargo policy for the embargo terms.

Cloud Infrastructure

The Collaboratory hosts an OpenStack cloud with nearly 1150 cores and over 3.2 PB of raw storage available for use by researchers. Users can launch and access Linux virtual machines running common distributions such as Ubuntu or CentOS through the OpenStack console. The Collaboratory offers multiple ways for researchers to share their tools including VM sharing through the OpenStack Console and the Dockstore. The latter is a developing service that offers Docker container sharing based on the GA4GH-compliant tool descriptor.
