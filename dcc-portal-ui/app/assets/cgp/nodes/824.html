Project Summary

This project aims to elucidate the comprehensive somatic changes in the genome (mutation, rearrangement and copy number change), the epigenome (methylation etc.) and transcriptome of virus (both HBV and HCV)-associated hepatocellular carcinoma as an international contribution.
